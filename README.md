# String Utils

A collection of string utilities:

* Hamming distance;
* Levenshtein Distance;
* Get XML value;
* Get JSON value from object;
* Normalization (UTF-8 -> HTML, UTF-8 -> ASCII, ecc.);
* Padding (rpad / lpad);
* Parse Separated Values (eg. CSV);
* Obfuscation.

## Build

- `git clone https://github.com/giosil/string-utils.git`
- `mvn clean install`
